# Flower-Shop-Project
ASP.NET Core MVC - Web Application for a Flower Shop.
## Used Technologies and NuGet Packages:

<hr/>
<ul>
<li>  .NET 6.0 </li>
<li>  ASP.NET Core MVC </li>
<li>  EntityFrameworkCore </li>
<li>  MSSQL Server </li>
<li>  Custom CSS </li>
<li>  Bootstrap </li>
</ul>
